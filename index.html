Description of the Project

Simple Extractive Text Summarization

This project implements extractive text summarization, a technique that generates summaries by selecting important sentences from the input text. It ranks sentences based on their importance using TF-IDF (Term Frequency-Inverse Document Frequency) for vectorization and cosine similarity to measure sentence similarity.

The backend is implemented in Python, leveraging its libraries for calculations. The frontend uses JavaScript, allowing the summarization to run directly in the web browser. This simple technique does not involve deep learning but effectively summarizes based on the input text.
